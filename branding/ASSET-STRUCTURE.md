## Required Asset Set Per Product Layer

Each layer must eventually include:

- primary horizontal logo;
- stacked logo;
- icon-only mark;
- dark-background lockup;
- light-background lockup;
- monochrome lockup;
- SVG source;
- transparent PNG exports;
- social card;
- presentation cover;
- website product card;
- animated reveal;
- watermark version.

## Rule

Generated posters are campaign assets, not canonical logos. A logo is accepted only after it has a simple vector master, small-size validation, monochrome version and usage rules.
