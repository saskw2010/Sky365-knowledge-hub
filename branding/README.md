# Sky365 Branding Architecture

> Canonical branding workspace for the Sky365 product family.

## Current Product Evolution

```text
Sky365 Core → Sky365 Enterprise → Sky365 AI → Sky365 Agentic → Skyverse
```

## Master Brand

**Sky365** is the master brand.

The product family is structured as five evolution layers:

1. **Sky365 Core** — Modular Monolith ERP
2. **Sky365 Enterprise** — Scale, Governance & Integration
3. **Sky365 AI** — Intelligence & Automation Layer
4. **Sky365 Agentic** — Autonomous Agents & Digital Workforce
5. **Skyverse** — Metadata-Driven Agentic Platform

## Key Rule

Sky365 Agentic is currently positioned immediately before Skyverse. Skyverse remains the highest evolution of the platform and unifies applications, agents, workflows, data, context, memory, analytics, social and experience layers.
