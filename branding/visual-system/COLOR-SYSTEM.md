# Sky365 Color System

| Token | Hex | Use |
|---|---:|---|
| Sky Navy | `#06101D` | Primary dark background |
| Sky Blue | `#168BFF` | Master-brand accent |

| White Cloud | `#F4FAFF` | Primary text on dark surfaces |

## Gradient Rules

- Core: blue → cyan
- Enterprise: green → teal
- AI: orange → amber
- Agentic: violet → magenta
- Skyverse: indigo → cyan → violet

## Accessibility

- Maintain WCAG AA contrast for text.
- Do not use glow as the only information carrier.
- Avoid edition colors for long body text.
- Always pair color with a label or icon.

## Consistency Rule

Edition colors are semantic identifiers. Do not swap them between assets without updating the brand decision record.
