ما العلوم الموجودة داخل SKY365؟

خريطة معرفية مستخلصة من الرؤية، الـblueprints، الأبحاث، القرارات ومسار Zvec. الطبقة الأولى اتجاهات علمية؛ الثانية محاور بحث؛ الثالثة مفاهيم وتقنيات وقرارات. ملفات GitHub تظهر فقط كدليل عند فتح العقدة.

طريقة القراءة: ابدأ باتجاه علمي، ثم افحص محاوره، ثم افتح المفاهيم. الخطوط الذهبية المتقطعة تكشف العلاقات العابرة للاتجاهات؛ مثل ارتباط Memory بالAgent Runtime، وارتباط Semantic Layer بالـERP، وارتباط Evaluation بالسلامة.
