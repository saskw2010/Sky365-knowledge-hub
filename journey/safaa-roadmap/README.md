# Safaa Canada Execution Roadmap

Landing page and visual execution guide covering:

- work-permit authorization follow-up;
- PEBC document and examination path;
- English and French examination cadence;
- language-training allocation;
- combined pharmacy and language learning;
- curated PEBC English resources;
- YouTube Auto Dubbing usage and limitations.

Open `index.html` through GitHub Pages for the visual experience.

## Visual format

Both roadmaps are rendered directly as responsive HTML/CSS. This keeps all Arabic text complete, searchable, editable and readable on mobile and wide screens without depending on fixed-size images.
