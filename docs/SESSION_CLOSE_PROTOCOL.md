# SKY365 Session Close Protocol

This protocol standardizes end-of-session documentation.

1. Session Summary
- Objectives
- Decisions
- New Ideas
- Architecture Changes
- Technical Findings
- Risks
- Future Recommendations

2. Architecture Decisions
Store/update files under `/docs/architecture/review-notes/`.

3. Discussion Backlog
Store/update files under `/docs/architecture/discussion-backlog/`.

4. Chat Archive
Store under `/docs/history/chat-archive/` in both HTML and Markdown.

5. Update indexes.

6. Detect new engines and update catalogs.

7. Generate next-session TODO.

8. Generate suggested commit message.
